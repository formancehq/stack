# ListAccountsBalanceOperator

Operator used for the filtering of balances can be greater than/equal, less than/equal, greater than, less than, equal or not.



## Values

| Name  | Value |
| ----- | ----- |
| `GTE` | gte   |
| `LTE` | lte   |
| `GT`  | gt    |
| `LT`  | lt    |
| `E`   | e     |
| `NE`  | ne    |
