# Formance Ledger v2

## What is Ledger?

Formance Ledger is a real-time money tracking microservice that lets you model and record complex financial transactions. It offers atomic, multi-posting transactions and is programmable using Numscript, a dedicated DSL (Domain Specific Language) to model and templatize such transactions.

Did you find a bug in Formance Ledger? Well, thanks! We need your help to make it better. You can [file a bug report](https://github.com/formancehq/ledger/issues/new?assignees=&labels=bug&template=bug_report.md&title=) and tell us all about it.

## Formance Ledger is Open Source

Formance Ledger is [Open Source](https://github.com/formancehq/ledger/blob/main/LICENSE), and we welcome your contributions, whether it's code or working on these very documents. We maintain a welcoming and inclusive community: [Join us on GitHub](https://github.com/formancehq/ledger) and learn how to get involved.
