In addition to binaries and distribution packages, Formance Platform services are also available as Docker images. These images are available as per listed below.

<Warning/>

## Docker Compose

The latest `docker-compose.yml` file can be found in the [Formance Platform repository](https://github.com/formancehq/stack/blob/main/docker-compose.yml)

## Individual Docker Images

| Name          | Image                                                  | Standalone Usage |
|---------------|--------------------------------------------------------|------------------|
| `ledger`      | `ghcr.io/formancehq/ledger`                            | ✅                 |
| `payments`    | `ghcr.io/formancehq/payments`                          | ✅                 |
| `auth-dex`    | `ghcr.io/formancehq/auth-dex`                          | ❌                 |
| `auth`        | `ghcr.io/formancehq/auth`                              | ❌                 |
| `control`     | `ghcr.io/formancehq/control`                           | ❌                 |
| `search`      | `ghcr.io/formancehq/search`                            | ❌                 |
| `webhooks`    | `ghcr.io/formancehq/webhooks`                          | ❌                 |
