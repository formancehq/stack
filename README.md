# Stack

Launch stack with `task run` of `docker-compose up -d`

## Base URLs
- Auth: http://localhost/api/auth
- Ledger: http://localhost/api/ledger
- Payments: http://localhost/api/payments
- Search: http://localhost/api/search
- Webhooks: http://localhost/api/webhooks
- Redpanda UI: http://localhost:8081
- OpenSearch Dashboard: http://localhost:5601
